Google Sans Flex
================

This site uses only the bundled variable font:

  GoogleSansFlex-VariableFont_GRAD,ROND,opsz,slnt,wdth,wght.ttf

The stylesheet uses `font-display: swap` so text renders immediately with a
system fallback while the font loads. The font is licensed under the SIL Open
Font License; see OFL.txt.
